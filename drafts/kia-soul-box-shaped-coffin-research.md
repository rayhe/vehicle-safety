# Research: The Kia Soul — Box-Shaped Coffin

## Angle
The Kia Soul exists in a classification no-man's-land — NHTSA calls it an SUV, IIHS calls it a small car/wagon, FARS classifies it as a sedan, and Kia marketed it as a quirky urban crossover with dancing hamster ads. The FARS data tells you which classification is correct: it dies like a subcompact.

## Kill Test
✅ Novel: No existing Kia Soul article. The classification chaos angle (three agencies, three classifications) is unexplored.
✅ Data-driven: Lethality of 0.643 vs SUV class average 0.524 — a 22.7% gap in the wrong direction for something NHTSA classifies as an SUV.
✅ Surprising: IIHS gave it Top Safety Pick+ (2018). FARS says 64.3% of its fatal crashes kill someone.
✅ Actionable: If you're shopping for a Soul because you think you're getting SUV safety in a small package, you're not.

## Novel Analysis: The Classification Tax
Original finding: The Soul's lethality (0.643) sits exactly between the sedan class average (0.645) and the SUV class average (0.524). It performs identically to a sedan despite being classified as an SUV by NHTSA. This means NHTSA's classification gives buyers a false sense of security — the "SUV" badge implies SUV-level crash protection that doesn't exist.

Calculation:
- Sedan class lethality: 0.645
- Soul lethality: 0.643
- Difference from sedan average: -0.002 (essentially identical)
- SUV class lethality: 0.524
- Gap from SUV average: +0.119 (22.7% worse)
